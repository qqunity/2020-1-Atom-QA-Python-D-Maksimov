- Для запуска скприпта необходимо указать через ключ -f путь к файлу с логами. Пример:
    ./log_processing -f nginx_logs.txt
- Для запуска без интерактивного режима используется ключ -s, а для указание команды ключ -cmd. Номер команды соответсвует порядковому номеру в гите. Для 2 команды используется специальный ключ -req_type для указания типа запроса. Пример:
    ./log_processing -f nginx_logs.txt -s -cmd 2 -req_type GET
- Для указание записи в файл используется ключ -o. Пример:
    ./log_processing -f nginx_logs.txt -s -cmd 1 -o filename
- Все ошибки записываются в err_log.txt.
- Файл sub_script необходим для корректной работы.
